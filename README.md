## 개발이유
사회 이슈에 따른 사람들의 관심도가 지역에 따라 어떻게 분포되는 지를 보여주기 위한 통계 서비스


## Osori 서비스
* 이슈 응답에 따라 전국 지역별 통계 분포를 볼 수 있어요.
* 한 지역의 시/군/구 통계 분포를 볼 수 있어요. 
* Osori에서 볼 수 있는 서비스
  - 남녀 응답수, 남녀 최다 응답 연령대
  - 전체 응답률, 남녀 전체 응답률
  
## Getting Start
### Client
> ```js
> npm install
> npm start
> ```
 ### Server
> ```js
> npm install
>  // .env_example 파일을 보고 .env 파일 생성및 기입
> npm run start:db // 설정한 mongodb에 더미데이터 생성시 
> npm start
> ```
